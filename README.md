## Sorting
어떤 데이터가 주어졌을 떄 순서대로 나열하는 것

### Bubble Sort
두 인접한 데이터를 비교 후, 앞 > 뒤 조건에 만족할 때 위치를 바꾸는 알고리즘 <- 처음부터 끝까지 이를 반복<br>
눈으로 보는 원리 : https://visualgo.net/en/sorting 

**분석**
반복문이 2개 이므로 보통 : O(n^2)<br>
완전 정렬이 되어있을 경우 : O(n)
